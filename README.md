[fastText](https://github.com/facebookresearch/fastText) Rust binding

## Installation

Add it to your ``Cargo.toml``:

```toml
[dependencies]
fasttext = "0.6"
```

Add ``extern crate fasttext`` to your crate root and your're good to go!
